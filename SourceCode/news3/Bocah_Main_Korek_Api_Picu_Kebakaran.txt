Bocah Main Korek Api Picu Kebakaran
Reporter : Nurito | Editor : Dunih | Senin, 21 April 2014 19:05 WIB | Dibaca 2784 kali
(Foto :  / Beritajakarta.com)
  Hati-hati mengawasi anak bermain, karena bisa jadi benda yang dimainkannya barang yang berbahaya. Seperti musibah terbakarnya sedan B 8832 GB yang hangus terbakar karena ulah anak-anak bermain korek api di bedeng rumah Jl Mawar Merah 7 RT 02/07, Malaka Jaya, Duren Sawit, Jakarta Timur, sekitar pukul 17.00. 

"  Informasi awal, kebakaran ini diduga akibat ada anak-anak kecil main 
korek api. Kemudian api menyambar tangki bensin hingga akhirnya terjadi 
kebakaran"

Saat kejadian sedan warna hijau tersebut sedang diparkir di sebuah bedeng berukuran 5x6 meter persegi. Di sekitar lokasi, sejumlah anak kecil berusia 5-8 tahun bermain korek api. Entah kenapa api tersebut sampai akhirnya menyambar mobil tersebut.Melihat mobil sudah terbakar, anak-anak yang diduga bermain korek api itu langsung kabur. Api cepat berkobar, hingga akhirnya membakar seluruh mobil dan bedeng tersebut. Warga sekitar, yang melihat kejadian langsung berupaya memadamkan api menggunakan alat seadanya. Namun, tak berhasil dan api malah terus berkobar. Bahkan sempat menimbulkan suara ledakan yang diduga dari tangki bensin.Pemilik mobil, Waskam (50) mengatakan, saat kejadian sedang berada di dalam rumahnya yang tak jauh dari lokasi kejadian. Ia mengaku baru mengetahui mobilnya terbakar dari tetangganya yang berteriak adanya kebakaran.Tukimin (50), warga sekitar mengatakan, awalnya melihat kepulan asap di sekitar bedeng. Semula ia mengira ada warga yang membakar sampah, mengingat bedeng berada persis di samping tanah lapang.Â  "Warga berupaya memadamkan api menggunakan air dari selokan, tapi tak berhasil. Api cepat berkobar dan membakar seluruh bagian mobil," ujarnya, Selasa (21/4).Namun saat itu, Tukimin mengaku tak melihat lagi anak-anak yang diduga bermain korek api di sekitar lokasi kejadian.Komandan Peleton (Danton) Pemadam Kebakaran dan Penanggulangan Bencana (Damkar dan PB) Duren Sawit, Sutrisno mengatakan, kebakaran diduga akibat sejumlah anak kecil yang bermain korek api. Api dalam hitungan detik langsung menyambar mobil hingga membakar seluruh bagiannya."Informasi awal, kebakaran ini diduga akibat ada anak-anak kecil main korek api. Kemudian api menyambar tangki bensin hingga akhirnya terjadi kebakaran," jelas Sutrisno.Untuk memadamkan kobaran api, pihaknya mengerahkan 12 unit mobil pemadam. Api berhasil dijinakkan petugas sekitar 30 menit setelah kejadian.
