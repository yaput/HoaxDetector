1.167 Siswa Paket C Ikut UNPK di Jaktim
Reporter : Jhon Syah Putra Kaban | Editor : Widodo Bogiarto | Senin, 14 April 2014 21:11 WIB | Dibaca 2009 kali
(Foto :  / Beritajakarta.com)
  Sebanyak 1.167 siswa peserta program Paket C pendidikan kesetaraan setingkat SMA tercatat mengikuti pelaksanaan Ujian Nasional Pendidikan Kesetaraan (UNPK) tahun 2014 di Jakarta Timur. Peserta UNPK ini terbagi menjadi dua, yaitu program IPA sebanyak 18 peserta dan program IPS sebanyak 1.657 peserta.

"Program Paket C ini menyasar kepada anak-anak atau orang yang belum 
selesai SMA, tapi sudah bekerja. Ada juga yang umurnya sudah melewati 
waktu sekolah  "

Kepala Suku Dinas Pendidikan Menengah (Dikmen) Jakarta Timur, Budiana menjelaskan, pada hari pertama peserta UNPK untuk Program Paket C jurusan IPS mengerjakan materi ujian Bahasa Indonesia dan Geografi. Sedangkan untuk jurusan IPA mengerjakan materi Bahasa Indonesia dan Biologi."Program Paket C ini menyasar kepada anak-anak atau orang yang belum selesai SMA, tapi sudah bekerja. Ada juga yang umurnya sudah melewati waktu sekolah," kata Budiana, Senin (14/4). Budiana mengatakan, ada enam mata pelajaran yang termasuk kategori UN, yaitu Biologi, Bahasa Indonesia, Matematika, Kimia, Fisika dan Bahasa Inggris. Sedangkan untuk Program Paket C ditambah pelajaran Pendidikan Kewarganegaraan (PKn). "Totalnya ada tujuh mata pelajaran yang di UN-kan," terangnya. Selain itu, Sudin Dikmen Jaktim telah mengangkat 170 orang sebagai pengawas UNPK program Paket C tersebut. Sedangkan penyelenggaraannya sendiri akan dilaksanakan selama tiga hari, terhitung mulai Senin (14/4) hingga Rabu (16/4).Menurutnya, kriteria kelulusan peserta UNPK apabila rata-rata nilai akhir, dari seluruh mata pelajaran yang diujikan paling rendah mencapai 5,5 dan nilai akhir setiap mata pelajaran paling rendah 4,0.Sementara itu, lokasi penyenggaraan UNPK Program Paket C itu diadakan di lima lokasi, yaitu SMP Negeri 255 Duren Sawit, SMP Negeri 167 Duren Sawit, SMP Negeri 27 Duren Sawit, SMP Negeri 195 Duren Sawit, dan di SMP Negeri 139 Duren Sawit.
