Rumah Mewah di Duri Kepa Terbakar
Reporter : TP Moan Simanjuntak | Editor : Agustian Anas | Senin, 28 April 2014 16:32 WIB | Dibaca 1933 kali
(Foto :  / Beritajakarta.com)
Peristiwa kebakaran masih menjadi momok menakutkan bagi warga Jakarta. Betapa tidak, hari ini Senin (28/4), sedikitnya ada tiga peristiwa kebakaran terjadi di ibu kota. Setelah tadi pagi, api membakar sebuah toko sepatu di Pasar Rumput, Kelurahan Pasar Manggis, Setiabudi, Jakarta Selatan dan 3 unit rumah di Kelurahan Kelapa Gading, siang harinya giliran rumah mewah berlantai tiga di perumahan Green Ville, Jl Mangga 24, No 165, RT 06/03, Kelurahan Duri Kepa, Kecamatan Kebon Jeruk yang dilalap si jago merah.

"  Kebakaran akibat korsleting listrik. Tiba-tiba api muncul dari kamar saya"

Kebakaran yang melanda rumah milik Leo atau yang akrab disapa Alin itu terjadi sekitar pukul 13.30. Api diduga berasal dari korsleting listrik di kamar pembantu yang berada di lantai dua.Â  Wati (20), pembantu di rumah tersebut mengatakan, kebakaran rumah majikannya itu diduga akibat korsleting listrik. "Kebakaran akibat korsleting listrik. Tiba-tiba api muncul dari kamar saya," ujar Wati.Melihat api berkobar, dia langsung berusaha memadamkan kobaran api dengan peralatan seadanya. Namun upayanya tidak berhasil, api justru semakin membesar hingga menghanguskan lantai dua tersebut.Api baru berhasil dipadamkan satu jam kemudian setelah 15 unit mobil pemadam kebakaran dari Suku Dinas Damkar dan PB Jakarta Barat dikerahkan ke lokasi. "Tidak ada korban jiwa. Kebakaran diduga akibat korsleting listrik," kata Abdul Wahid, Kepala Seksi Operasional Sudin Damkar dan PB   Jakarta Barat.
