Sekolah Diamuk Api, Kelas dan Lab Terbakar
Reporter : Budhi Firmansyah Surapati | Editor : Dunih | Sabtu, 03 Mei 2014 19:30 WIB | Dibaca 1637 kali
(Foto :  / Beritajakarta.com)
Musibah kebakaran melanda SMPN 200, Jl Rorotan Simpang 3, RW 11, Kelurahan Rorotan, Kecamatan Cilincing, Jakarta Utara. Akibat peristiwa tersebut, 2 ruang kelas, 1 ruang lab IPA, 1 gudang dan kantin sekolah hangus terbakar.

"Api membesar dengan cepat karena ada lab IPA. Kita menduga banyaknya cairan yang mudah terbakar menyebabkan api cepat membesar  "

Kejadian tersebut terjadi sekitar pukul 17.30 yang diduga berasal dari hubungan arus pendek listrik. Untuk memadamkan api, 10 unit mobil pemadam kebakaran dikerahkan ke lokasi. Beruntung pukul 17.55 api sudah dapat dikuasai. Walau tidak sampai memakan korban jiwa, kerugian yang diderita pihak sekolah mencapai Rp 300 juta.Afif (31) warga RT 01/11, Kelurahan Rorotan, mengatakan, kebakaran terjadi menjelang Maghrib. Bersama warga lain ia berupaya memadamkan api dengan alat seadanya. Namun, upaya tersebut tidak berhasil."Kita berupaya memadamkan, tapi api sudah keburu membesar," ujarnya, Sabtu (3/5).Sementara itu, Perwira Piket Sudin Pemadam Kebakaran dan Penanggulangan Bencana (Damkar dan PB) Jakarta Utara, Latif Ahmad mengaku, laporan diterima pihaknya sekitar pukul 17.30 tadi. Beruntung, karena hari ini akhir pekan, jalanan tidak macet sehingga pihaknya cepat tiba di lokasi." Api membesar dengan cepat karena ada lab IPA. Kita menduga banyaknya cairan yang mudah terbakar menyebabkan api cepat membesar ," jelasnya.Mengenai penyebab kebakaran, Latif menduga karena arus pendek yang berasal dari sebuah kabel di luar ruang gudang sekolah. Kebetulan gudang sedang penuh berisi bangku dan kursi bekas yang terbuat dari kayu sehingga materialnya mudah terbakar."Kerugian kita taksir sekitar Rp 300 juta. Tidak ada korban luka maupun korban jiwa," tandasnya.
