3 Rumah di Kelapa Gading Ludes Terbakar
Reporter : Budhi Firmansyah Surapati | Editor : Dunih | Senin, 28 April 2014 14:30 WIB | Dibaca 955 kali
(Foto : Budhi Firmansyah Surapati / Beritajakarta.com)
Kebakaran kembali terjadi DKI Jakarta. Kali ini sebanyak 3 unit rumah di Jl Puskesmas, RT 06/06, Kelurahan Kelapa Gading Timur, Kecamatan Kelapa Gading, Jakarta Utara, ludes dilalap si jago merah. Beruntung dalam peristiwa itu, tidak ada warga yang menjadi korban jiwa. 

"Dugaan awal api berasal dari korsleting listrik. Kerugian ditaksir sekitar Rp 50 juta  "

Kejadian kebakaran sendiri diketahui warga sekitar pukul 11.30. Saat itu, warga dikagetkan dengan asap yang membesar dari salah rumah kontrakan milik Sihombing (50). Namun, karena api cepat membesar, warga kewalahan sehingga tidak dapat memadamkan api yang langsung menghanguskan 3 unit rumah.Ketua RT 06/06, Kelurahan Kelapa Gading Timur, Nur Ali Ramlan (65) mengatakan, awalnya warga melihat asap cukup besar dari rumah kontrakan yang ditempati Yanto. Kemudian, sebagian warga berupaya memadamkan api dengan peralatan seadanya."Kita coba padamkan, tapi karena angin cukup besar api jadi membesar dan merembet ke-2 rumah lain milik Pak Sihombing. Melihat api membesar ada warga yang lapor ke pemadam," ujarnya, Senin (28/4).Menurut Nur, ketiga rumah yang terbakar tersebut milik dari Sihombing (60). Satu unit merupakan rumah utama yang ditempati oleh pemilik, sedangkan dua unit rumah lainnya dikontrakkan."Pada saat kejadian rumah itu kosong semua. Awalnya asap terlihat dari kontrakan yang ditempati Yanto (40)," ucapnya.Sementara itu, Kepala Seksi Opersional Sudin Pemadam Kebakaran dan Penanggulangan Bencana (Damkar dan PB) Jakarta Utara, Muchtar Zakaria mengatakan, untuk memadamkan api pihaknya menerjunkan sebanyak 16 unit mobil Damkar. Setelah berjibaku dengan api, kebakaran dapat dikuasai sekitar satu jam kemudian." Dugaan awal api berasal dari korsleting listrik. Kerugian ditaksir sekitar Rp 50 juta ," tandasnya.
