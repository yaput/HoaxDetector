Pabrik Celana Jins di Srengseng Terbakar
Reporter : Rio Sandiputra | Editor : Agustian Anas | Rabu, 23 April 2014 10:51 WIB | Dibaca 1491 kali
(Foto : Rio Sandiputra / Beritajakarta.com)
Si jago merah kembali mengamuk di wilayah Jakarta Barat. Sebuah pabrik celana jins di Jalan Swadharma Raya No 69 B, Srengseng, Jakarta Barat, ludes dilalap api, Rabu (23/4). Karyawan pabrik panik dan langsung berhamburan keluar. Tidak ada korban jiwa dalam peristiwa tersebut. Namun kerugian materi diperkirakan mencapai ratusan juta rupiah. 

"  Saya melihat asap mengepul dari dalam sekitar pukul 07.20 disusul kobaran api"

Peristiwa kebakaran terjadi sekitar pukul 07.20. Api diduga berasal dari korsleting pada alat pengering. "Saya melihat asap mengepul dari dalam sekitar pukul 07.20 disusul kobaran api," ujar Mulyana, pemilik warung kopi yang berada di seberang pabrik tersebut, Rabu (23/4).Karyawan pabrik beserta warga bahu membahu memadamkan kobaran api. Namun upaya tersebut tidak berhasil dan api justru semakin membesar. Beruntung, petugas pemadam dari Suku Dinas Pemadam Kebakaran dan Penanggulangan Bencana (Sudin Damkar dan PB) Jakarta Selatan dan Jakarta Barat cepat tiba di lokasi sehingga api tidak merembet ke bangunan lain. "Banyak bahan-bahan celana sehingga kita lokalisir terlebih dahulu, agar api tidak membesar dan meluas," ungkap Edi Sucipto, Kepala Seksi Operasi Suku Dinas Pemadam Kebakaran dan Penanggulangan Bencana Jakarta Selatan.Sebanyak 26 unit mobil pemadam kebakaran dikerahkan untuk memadamkan kobaran api. Setelah berjibaku selama satu jam, akhirnya petugas berhasil menjinakkan kobaran api. "Api sudah dapat dikuasai. Saat ini sedang dilakukan pendinginan," katanya. Edi menjelaskan, dari keterangan karyawan, kebakaran akibat korsleting pada alat pengering. "Dari keterangan karyawan, kebakaran akibat   konsleting pada alat pengering. Lokasi yang terbakar 10x50 meter," ungkapnya.
