Toko Sepatu di Pasar Rumput Terbakar
Reporter : Rio Sandiputra | Editor : Agustian Anas | Senin, 28 April 2014 09:46 WIB | Dibaca 1610 kali
(Foto : Rio Sandiputra / Beritajakarta.com)
Si jago merah kembali mengamuk di Ibu Kota. Setelah menghanguskan ribuan kios di Pasar Senen, Jakarta Pusat, kini giliran Pasar Rumput yang terletak di Jalan Sultan Agung, Kelurahan Pasar Manggis, Kecamatan Setiabudi, Jakarta Selatan yang terbakar. 

"  Kejadiannya sekitar pukul 05.15. Api belum kelihatan tapi asap tebal terlihat membubung tinggi ke atas"

Peristiwa kebakaran terjadi Senin (28/4) sekitar pukul 05.15. Api membakar sebuah toko sepatu milik Buyung di lantai 2 Blok A. "Kejadiannya sekitar pukul 05.15. Api belum kelihatan, tapi asap tebal terlihat membubung tinggi ke atas," kata Edo Warsi, Kepala Pasar Rumput, Senin (28/4).Kebakaran terjadi saat toko masih dalam keadaan tutup. "Toko belum ada yang buka sama sekali karena memang masih pagi. Saya melihat kabel-kabel listrik sudah terbakar dan sempat ada ledakan," ungkapnya.Melihat kejadian itu, dia bersama petugas keamanan langsung mematikan gardu listrik induk di pasar tersebut. "Aliran listrik dari gardu induk pasar langsung saya matikan agar tidak merembet ke toko yang lain. Sedangkan petugas keamanan langsung menghubungi petugas pemadam," ungkapnya. Petugas pemadam yang mendapat informasi kebakaran langsung menerjunkan 33 unit mobil pemadam ke lokasi kejadian. Begitu tiba, p  etugas langsung melokalisir titik api."Kita langsung lokalisir titik api karena tokonya berada di tengah. Kita turunkan 33 mobil untuk mencegah kebakaran seperti yang terjadi di Pasar Senen kemarin," ungkap Edi Sucipto, Kepala Seksi Operasi Suku Dinas Pemadam Kebakaran dan Penanggulangan Bencana Jakarta Selatan.Berkat kesigapan petugas, api tidak merembet ke toko lain. "Syukur bisa kita tangani dengan cepat agar api tidak merembet. Api berhasil dikuasai 45 menit kemudian, tidak ada korban jiwa," ucapnya.Namun untuk penyebab kebakaran masih belum diketahui secara pasti. "Tapi dari keterangan saksi, ada yang melihat kabel-kabel terbakar. Biar nanti polisi yang menyelidiki lebih lanjut," tandasnya.
