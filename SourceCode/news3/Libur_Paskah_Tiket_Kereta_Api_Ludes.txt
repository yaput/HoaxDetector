Libur Paskah, Tiket Kereta Api Ludes
Reporter : Suriaman Panjaitan | Editor : Widodo Bogiarto | Kamis, 17 April 2014 13:47 WIB | Dibaca 3002 kali
(Foto : Yopie Oscar / Beritajakarta.com)
  Sebagaimana berulangkali terjadi pada hari libur di akhir pekan, tiket kereta api terjual ludes menjelang libur Hari Raya Paskah, Jumat (18/4). Tiket kereta api dari Jakarta tujuan Purwokerto, Yogyakarta, Solo, Semarang, dan Surabaya habis terjual. Sebanyak 26.000 tiket dari 56 perjalanan di wilayah Daerah Operasional 1 Jakarta untuk Kamis (17/4) dan Jumat (18/4) sudah dipesan calon penumpang sejak 90 hari sebelum perjalanan. 

"  Pas saya datang, langsung ke tempat pembelian tiket, ternyata tiket buat besok sudah habis terjual semuanya"

Sejumlah warga yang tidak kebagian tiket KA terpaksa menelan kecewa karena batal menikmati liburan panjang. Nita (25), calon pembeli tiket tujuan Yogyakarta mengaku, sudah datang ke Stasiun Senen sejak pukul 08.00. Namun ketika sampai di loket stasiun, ada pengumuman yang menyebutkan tiket kereta api telah habis."Pas saya datang, langsung ke tempat pembelian tiket, ternyata tiket buat besok sudah habis terjual semuanya," kata Nita, Kamis (17/4).Hal senada juga diungkapkan Sudarminto(56). Pria asal Tegal (Jawa Tengah) ini juga harus menahan rasa rindunya ketika mengetahui tiket kereta api untuk keberangkatan Jumat (18/4) sudah habis terjual. "Ya mau bagaimana lagi, saya terpaksa harus telepon saudara di kampung, kalau saya tidak jadi pulang besok lantaran tidak dapat tiket," ujarnya.Kepala Stasiun Senen, Supendi menjelaskan, tiket kereta api yang sudah habis terjual tidak hanya berlaku buat keberangkatan Jumat namun juga hari ini. "Dari kemarin tiket sudah habis terjual. Untuk hari ini tiket yang sudah terjual sebanyak 12.096 lembar, sedangkan Jumat, tiket yang sudah terjual itu sebanyak 12.801 lembar. Semua tiket yang terjual itu sesuai dengan kapasitas penumpang kereta," jelasnya.Supendi menambahkan, untuk keberangkatan hari ini, dari 21 kereta api yang ada, ada dua kereta api yang mengalami penambahan penumpang, yaitu kereta Kutojaya Utara jurusan Kutoarjo bertambah dari biasanya 846 penumpang menjadi 876 penumpang dan kereta Progo jurusan Lempuyangan Yogyakarta, yang biasanya sebanyak 848 penumpang menjadi 874 penumpang.Sedangkan untuk Jumat, penambahan penumpang terjadi pada kereta Bengawan dengan tujuan Purwosari, pada hari biasa hanya berjumlah 848 orang sedangkan untuk besok penambahan penumpang menjadi 904 orang."Penambahan kuota kursi kereta bisa terjadi apabila ada perintah dari manajemen PT KAI," tukasnya.
