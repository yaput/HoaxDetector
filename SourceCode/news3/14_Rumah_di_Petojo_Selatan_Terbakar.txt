14 Rumah di Petojo Selatan Terbakar
Reporter : Suriaman Panjaitan | Editor : Agustian Anas | Jumat, 02 Mei 2014 10:34 WIB | Dibaca 1504 kali
(Foto :  / Beritajakarta.com)
Satu minggu pascakebakaran di Pasar Senen, si jago merah kembali mengamuk di wilayah Jakarta Pusat. Sebanyak 14 rumah di Jalan Tanah Abang Lima Gang IV RT 002/02, Kelurahan Petojo Selatan, Kecamatan Gambir, ludes dilalap api, Jumat (2/5) dinihari. 

"  Api berasal dari rumah Pak Burhan. Api begitu cepat membesar"

Kebakaran yang melanda pemukiman padat penduduk itu terjadi sekitar pukul 00.05. Api diduga berasal dari hubungan pendek arus listrik dari rumah warga bernama Burhan.Warga yang mengetahui ada kebakaran bahu membahu memadamkan kobaran api dengan peralatan seadanya. Namun upaya warga tidak berhasil. Api begitu cepat membesar sehingga warga tidak berhasil menyelamatkan harta benda milik mereka. "Api berasal dari rumah Pak Burhan. Api begitu cepat membesar," kata Anton, Ketua RT 02 yang rumahnya juga ludes dilalap api. Api baru berhasil dipadamkan sekitar pukul 01.00 setelah 22 unit mobil pemadam kebakaran dikerahkan ke lokasi. "Ada 14 rumah yang terbakar. Api sudah berhasil dikuasai ," ujar Sudarno, Kasie Operasional Sudin Damkar dan PB Jakarta Pusat, Jumat (2/5). Dia mengatakan, kebakaran diduga akibat hubungan pendek arus listrik  . "Dugaan sementara karena korsleting listrik. Tidak ada korban jiwa, namun kerugian materi diperkirakan mencapai Rp 600 juta," ungkapnya.
