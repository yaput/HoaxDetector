Program tanpa DP Rumah Rakyat Anies-Sandi Dipermasalahkan tapi Tanpa DP dari Pengembang Dibiarkan
JAKARTA (voa-islam.com)- Politisi, Indra J Piliang menyatakan bahwa program Anis-Sandi untuk rumah tanpa DP sudah sangat jelas kegunaannya, juga sasarannya. Indra bahkan berani mengatakan bahwa program cagub yang masuk ke putaran kedua ini sebagai program untuk rakyat tanpa ada keuntungan konglomerasi, apalagi para pengemplang pajak.

“DP buat rumah. Bukan DP buat taipan-taipan pengemplang dana BLBI yang bisa tingkatin cadangan alutsista hingga nomor satu di ASEAN,” tulisnya, melalui akun Twitter pribadinya.

Apabila masyarakat memperhatikan, bahwa bukan hanya program seperti Anies-Sandi saja yang seperti itu. Pihak lain seperti konglomerat juga mempunyai program demikian.
“Mau iklan ginian? Banyak. Temasuk hunian apartemen di Pluit, Kelapa Gading, hingga kawasan-kawasan non aborigin lainnya. Mainkan. Cc: BI.”
Bila saja program ini dipermasalahkan tetapi untuk para pengembang dibiarkan, maka ia pun sangsi kepada Bank Indonesia soal keadilan.
“Buat pengembang, DP 0 persen itu wajar. Buat rakyat banyak? DP 0 persen itu melanggar aturan. BI berpihak kepada siapa? Konsorsium BLBI?” (Robi/voa-islam.com)
