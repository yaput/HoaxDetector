Ahok Sindir Tidak Miliki Program Kerja, Ini Jawaban Skak Mat Yusril
JAKARTA (voa-islam.com)- Basuki Tjahaja Purnama atau Ahok mempertanyakan pakar hukum tata negara, Yusril Ihza Mahendra sebagai bacagub yang isunya tidak memiliki program kerja. Ketum PBB ini pun langsung merespon pertanyaan Ahok tersebut.
Yusril misalnya saja menyebut dan mengakui bahwa ia memang tidak memiliki program seperti menggusur warga miskin, sengsarakan warga, tidak membeli tanah yang sudah dimiliki DKI, dan lainnya.

Berikut celoteh Yusril untuk Ahok di akun Twitter pribadi miliknya: "Ahok bilang saya nggak punya program itu memang bener, bukan fitnah. Pertama saya gak punya program gusur rakyat miskin. Ahok punya;

Kedua saya gak punya program sengsarakan warga Kampung Aquarium. Ahok punya dan sudah dilaksanakan; Saya gak punya program mau gusur kampung Luar Batang, Ahok punya. Sudah mau dilaksanakan tapi gagal.
Saya gak punya program untuk beli tanah RS Sumberwaras, Ahok punya. Sudah dilaksanakan dan kini bermasalah. Saya gak punya program beli tanah milik DKI sendiri di Cengkareng. Sudah dilaksanakan dan bermasalah.
Saya tidak punya program seperti itu; Saya gak punya program bentak-bentak ibu-ibu. Ahok punya dan sudah dilaksanakan. Ahok punya program tidak boleh motong hewan qurban kecuali di tempat jagal hewan. Saya gak punya.
Itu deh kira2 yang saya gak punya program tapi Ahok punya, hehehe..." (Robi/voa-islam.com)
