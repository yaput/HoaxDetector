Ruko Dua Lantai di Pulomas Ludes Terbakar
Reporter : Nurito | Editor : Lopi Kasim | Rabu, 05 Februari 2014 20:01 WIB | Dibaca 719 kali
(Foto :  / Beritajakarta.com)
                    		Di tengah banjir yang menggenangi sejumlah kawasan di Jakarta Timur, Ruko Maju Karya Utama, di Blok III nomor 2, Kompleks Ruko Pulomas, Pulogadung, ludes dilahap si jago merah, sekitar pukul 16.45. Diduga, kebakaran dipicu sebuah lilin yang terjatuh dan mengenai karpet di dalam ruko, lantaran saat kejadian lokasi sekitar ruko sedang mengalami pemadaman listrik. Sebanyak 16 unit mobil pemadam kebakaran dikerahkan Suku Dinas Pemadam Kebakaran dan Penanggulangan Bencana (Damkar dan PB) Jakarta Timur ke lokasi. Api berhasil dipadamkan satu setengah jam kemudian.Informasi di lapangan, saat peristiwa kebakaran terjadi di dalam ruko masih ada pemilik dan sejumlah karyawan. Mendadak mereka dikejutkan dengan kobaran api dari bagian belakang ruko berlantai dua. Sejumlah karyawan langsung mencoba memadamkan kobaran api menggunakan air genangan yang ada di depan ruko. Namun, upaya tersebut tak membuahkan hasil. Sebab api masih terus berkobar, bahkan menjalar ke lantai dua. Banyaknya benda yang mudah terbakar membuat api sulit dipadamkan. Herta (32), pemilik ruko mengatakan, saat kejadian aliran listrik sedang dipadamkan. Sebab, saat itu kawasan ruko tengah dilanda banjir sejak pagi hari. Ia menduga ada anak buahnya yang menyalakan lilin namun kurang diawasi. Sehingga lilin menimpa karpet dan membakarnya."Listrik kan padam dari siang. Saya juga bingung kok sampai terjadi kebakaran begini. Mungkin ada anak buah saya yang menyalakan lilin," ujar Herta, Rabu (5/2).Kasie Operasi Sudin Damkar dan PB Jakarta Timur, Mulyanto mengatakan, untuk memadamkan kobaran api, pihaknya mengerahkan 16 unit mobil pemadam. Api baru berhasil dijinakkan petugas sekitar pukul 19.15. "Dugaan sementara kebakaran ini dari lilin. Karena aliran listrik di kompleks ruko sudah dipadamkan sejak siang," ucap Mulyanto.Pihaknya, sempat kewalahan memadamkan kobaran api. Sebab, seluruh ruangan sangat gelap dan dipenuhi asap tebal dari karpet yang terbakar. Dalam pemadaman itu pihaknya juga menggunakan 12 unit alat bantu pernafasan bagi anggotanya, untuk dapat menerobos ke dalam kobaran api. Selain itu, dua unit blower masing-masing berkekuatan 25 PK, dikerahkan untuk mengeluarkan asap tebal dari dalam ruangan.
