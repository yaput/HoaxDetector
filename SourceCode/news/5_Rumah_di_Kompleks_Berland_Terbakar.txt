5 Rumah di Kompleks Berland Terbakar
Reporter : Nurito | Editor : Dunih | Kamis, 30 Januari 2014 21:17 WIB | Dibaca 841 kali
(Foto :  / Beritajakarta.com)
                    Sedikitnya lima rumah di Jl Kesatriaan 8, RT 16/03 Kompleks Berland, Kebon Manggis, Matraman, Jakarta Timur, ludes dilalap si jago merah, sekitar pukul 19.00. Akibat kebakaran tersebut, seorang warga mengalami luka bakar di kedua kakinya hingga harus dilarikan ke rumah sakit terdekat.Informasi di lapangan, api pertama kali berkobar dari dalam rumah yang dijadikan warnet milik Lilik Mardiyanto (45). Dalam sekejap api langsung menyambar ke bagian instalasi listrik, hingga akhirnya terjadi kebakaran hebat. Sialnya Khaerul (18), pengunjung yang sedang berada di dalam warnet, tak sempat menghindar hingga mengalami luka bakar di kedua kakinya. "Saya lagi di dalam rumah, tahu-tahu warga pada teriak ada kebakaran. Pas saya keluar, tahu-tahu api sudah membesar dari warnet itu," ujar Suryati (40), warga sekitar, Kamis (30/1).Warga yang melihat kejadian tersebut, langsung berupaya memadamkan kobaran api. Namun, api sulit dipadamkan. Sehingga dalam hitungan menit api sudah menjalar ke bangunan yang ada di sampingnya hingga menghanguskan empat rumah lainnya. Masing-masing milik Marni (40), Yati (38), Marno (43) dan Darso (40).Kasie Operasi Sudin Pemadam Kebakaran dan Penanggulangan Bencana (Damkar dan PB) Jakarta Timur, Mulyanto mengatakan, untuk memadamkan kobaran api, pihaknya mengerahkan 27 unit mobil pemadam kebakaran. Ia sendiri belum mengetahui penyebab kebakaran tersebut. Namun, ia menduga kebakaran dipicu hubungan arus pendek listrik dari dalam warnet."Lokasinya sulit dijangkau, karena berada di dalam kompleks yang padat dan jalannya sempit. Api berhasil kita padamkan sekitar 45 menit setelah kejadian," jelas Mulyanto.
