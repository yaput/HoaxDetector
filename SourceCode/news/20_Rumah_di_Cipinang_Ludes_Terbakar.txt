20 Rumah di Cipinang Ludes Terbakar
Reporter : Nurito | Editor : Widodo Bogiarto | Selasa, 10 Maret 2015 14:14 WIB | Dibaca 2029 kali
(Foto : Nurito / Beritajakarta.com)
Sebanyak 20 rumah di Gang Sodong Timur 3 dan 4 RT 04/17, Kelurahan Cipinang, Pulogadung, Jakarta Timur, Selasa (10/3), ludes terbakar. Dugaan awal api berasal dari korsleting listrik.

"Tidak ada korban jiwa dalam kebakaran ini, tapi kerugian ditaksir mencapai Rp 1 miliar. Dugaan awal penyebab kebakaran akibat korsleting listrik  "

Informasi yang dikumpulkan menyebutkan, api pertama kali terlihat dari rumah seorang warga bernama Lasmini (57), sekitar pukul 11.00. Akibat hembusan angin kencang serta rumah yang berdempetan membuat api dengan cepat merembet.  Sobar (40), salah satu saksi mata menuturkan, api awalnya membakar lantai dua rumah Lasmini. Namun bukannya menyelamatkan diri, Lasmini justru sibuk mengevakuasi barang-barang miliknya.Warga kemudian langsung bergotong royong dengan peralatan seadanya berusaha memadamkan api. Namun upaya warga kurang berhasil karena api dengan cepat meludeskan 20 rumah yang umumnya berbentuk semi permanen. Akibat kebakaran itu, sebanyak 60 kepala keluarga (KK) terpaksa mengungsi.  "Saat api sudah membesar, Bu Lasmini malah sibuk ngangkutin barang-barang miliknya. Kami kemudian memaksa dia untuk menyingkir agar tidak menjadi korban," kata Sobar. Kepala Seksi Operasi Suku Dinas Penanggulangan Kebakaran dan Penyelamatan Jakarta Timur, Moelyanto mengungkapkan, untuk memadamkan api, pihaknya menerjunkan 20 unit mobil pemadam kebakaran. "Tidak ada korban jiwa dalam kebakaran ini, tapi kerugian ditaksir mencapai Rp 1 miliar. Dugaan awal penyebab kebakaran akibat korsleting listrik," jelas Moelyanto.
