Sedan Peugeot Terbakar di Exit Tol Sunter
Reporter : Budhi Firmansyah Surapati | Editor : Agustian Anas | Senin, 17 Februari 2014 09:15 WIB | Dibaca 892 kali
(Foto :  / Beritajakarta.com)
                    		Sebuah mobil Peugeot 207 bernomor polisi B 68 JD hangus terbakar di pintu keluar Tol Sunter arah Tanjung Priok di Jalan Ir Wiyoto Wiyono, Jakarta Utara, Senin (17/2) sekitar pukul 06.10. Beruntung, tidak ada korban luka maupun jiwa dalam insiden ini. Petugas Satwil Lantas Jakarta Utara, Aiptu Sumarsono, mengatakan, saat kejadian mobil yang dikendarai Andi Saleh ini tengah melaju dari arah Cawang dan berniat keluar di exit tol Sunter. Namun karena melihat percikan api dari kap mesin mobil, dia langsung menghentikan laju kendarannnya. "Pengemudi tiba-tiba melihat percikan api, dan menghentikan laju kendaraannya. Namun begitu keluar dari mobil, api sudah membesar sehingga ia menjauh dari kendaraannya," kata Aiptu Sumarsono.Dalam sekejap, api langsung membesar dan melahap hampir seluruh bagian mobil. Kebakaran tersebut sempat membuat kemacetan yang cukup panjang di ruas tol menuju Tanjung Priok arah Cawang. Sebanyak dua unit mobil pemadam kebakaran dari Sudin Damkar dan PB Jakarta Utara dikerahkan ke lokasi untuk memadamkan kobaran api. Kanit Laka Satwil Lantas Jakut, AKP Daud Iskandar, mengatakan bahwa pengemudi diketahui bernama Andi Saleh. Saat ini mobil sudah dievakuasi ke Mapolres Jakut."Kita belum dapat menyimpulkan penyebabnya. Kasusnya ditangani oleh Polsek Metro Tanjung Priok," ungkapnya.
