Pabrik Mebel di Makasar Terbakar
Reporter : Nurito | Editor : Erikyanri Maulana | Jumat, 06 Maret 2015 14:06 WIB | Dibaca 1430 kali
(Foto : Asep Kurnia / Beritajakarta.com)
Peristiwa kebakaran kembali terjadi di ibu kota. Kali ini si jago merah mengamuk dan menghanguskan pabrik mebel rumahan di RT 09/05 Cipinang Melayu, Makasar, Jakarta Timur, Jumat (6/3) sekitar pukul 11.30. Tidak ada korban jiwa maupun luka dalam peristiwa tersebut. Kebakaran ini juga ikut menghanguskan enam rumah yang berada di sekitar pabrik tersebut. 

"Warga kesulitan memadamkan api karena yang terbakar tiner dan cat  "

Informasi yang dihimpun menyebutkan, kobaran api kali pertama berkobar dari pabrik mebel rumahan milik Yoyo (35). Saat kejadian, kondisi pabrik tengah sepi ditinggal pemilik dan para pekerjanya menunaikan shalat Jumat. Diduga api berasal dari ruang penyimpanan tiner. Menurut keterangan salah seorang saksi mata, sempat terdengar ledakan sebanyak tujuh kali.   "Api cepat membesar dan merembet ke bangunan lain. Warga kesulitan memadamkan api karena yang terbakar tiner dan cat," ujar Rahmat, warga setempat, Jumat (6/3).Melihat kobaran api, warga lainnya langsung bahu membahu memadamkan kobaran api menggunakan peralatan seadanya. Sementara warga lainnya terlihat sibuk mengevakuasi barang-barang berharga miliknya ke lokasi yang lebih aman. Sebanyak 23 mobil dikerahkan untuk memadamkan kobaran api. Petugas pemadam kebakaran (damkar) yang tiba di lokasi kejadian juga sempat kesulitan memadamkan api karena banyaknya drum berisi tiner yang terbakar. "Dugaan sementara kebakaran akibat korsleting listrik. Petugas sempat kesulitan memadamkan api lantaran sumber air yang minim," ujar Moelyanto, Kasie Operasi Sudin Pemadam Kebakaran dan Penanggulangan Bencana (Damkar dan PB) Jakarta Timur. Ditambahkan Moelyanto, setelah berjibaku memadamkan api, petugas akhirnya bisa memadamkan kebakaran tersebut dalam waktu dua jam.
