Showroom Mobil di Kebon Jeruk Terbakar
Reporter : TP Moan Simanjuntak | Editor : Erikyanri Maulana | Minggu, 02 Maret 2014 16:18 WIB | Dibaca 565 kali
(Foto :  / Beritajakarta.com)
                    Peristiwa kebakaran kembali terjadi di ibu kota. Kali ini si jago merah mengamuk dan menghanguskan bangunan showroom mobil berlantai dua di Perumahan Sunrise Garden, Jl Mutiara III, Blok G, Kedoya Selatan, Kebon Jeruk. Beruntung, tidak ada korban jiwa maupun luka dalam peristiwa kebakaran tersebut. Hanya saja, kerugian akibat kebakaran itu ditaksir mencapai ratusan juta rupiah. Arman (35), warga setempat menuturkan, kobaran api yang membakar bangunan showroom mobil tersebut mulai terlihat sekitar pukul 13.45. "Saat kejadian, semua dalam kondisi panik. Karyawan berusaha memadamkan api sekaligus mengevakuasi mobil yang ada di dalam showroom. Tapi karena angin bertiup kencang, api sulit dipadamkan," ujar Arman, Minggu (2/3).Kasie Operasional Sudin Pemadam Kebakaran dan Penanggulangan Bencana (Damkar dan PB) Jakarta Barat, Abdul Wahab menuturkan, untuk memadamkan kobaran api, pihaknya menerjunkan 14 mobil pemadam kebakaran. Terlebih, lokasi showroom mobil ini berada di tengah-tengah pemukiman warga. Beruntung, api akhirnya berhasil dipadamkan dalam waktu satu jam kemudian. "Saat ini kami masih melakukan proses pendinginan. Untuk penyebabnya, kami menduga dari korsleting listrik. Tidak ada korban jiwa dalam musibah kebakaran ini," tandasnya.
