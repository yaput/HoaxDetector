Api di Wisma Kosgoro Sudah Padam
Reporter : Rudi Hermawan | Editor : Agustian Anas | Selasa, 10 Maret 2015 06:46 WIB | Dibaca 1625 kali
(Foto :  / Beritajakarta.com)
Api yang membakar gedung Wisma Kosgoro di Jalan MH Thamrin, Menteng, Jakarta Pusat, pada Senin (9/3) pukul 18.30, akhirnya berhasil dipadamkan petugas Dinas Penanggulangan Kebakaran dan Penyelamatan DKI Jakarta, Selasa (10/3) diinihari sekitar pukul 02.00 WIB.

"  Api berhasil kami kuasai sekitar pukul 02.00 WIB dan selanjutnya dilakukan pendinginan"

"Api berhasil kami kuasai sekitar pukul 02.00 WIB dan selanjutnya dilakukan pendinginan," kata Subejo, Kepala Dinas Penanggulangan Kebakaran dan Penyelamatan DKI Jakarta di lokasi, Selasa (10/3).Pihaknya mengerahkan 38 mobil pemadam kebakaran untuk memadamkam api yang menghanguskan lima lantai gedung Wisma Kosgoro, yakni lantai 16 hingga 20."Kami kerahkan 38 unit mobil pemadam untuk memadamkan kobaran api dan api berhasil dikuasai pukul 02.00 dinihari. Tidak ada korban jiwa dalam insiden tersebut," ujarnya.Sementera itu, Kepala Suku Dinas Penanggulangan Kebakaran dan Penyelamatan Jakarta Pusat, Idris, belum bisa memastikan penyebab serta kerugian yang diderita akibat insiden tersebut."Penyebab kebakaran masih diselidiki aparat kepolisian.   Dugaan sementara akibat korsleting listrik," ungkapnya.
