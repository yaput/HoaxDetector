3 Ruko di Cengkareng Ludes Terbakar
Reporter : TP Moan Simanjuntak | Editor : Dunih | Senin, 03 Februari 2014 13:05 WIB | Dibaca 585 kali
(Foto :  / Beritajakarta.com)
                    Musibah kebakaran menghanguskan tiga unit ruko berlantai dua di Jl Sakura, Sumur Bor, Kelurahan Cengkareng Barat, Kecamatan Cengkareng, Jakarta Barat. Beruntung tidak ada korban jiwa dalam kebakaran tersebut.Nurdin (43) warga RW 02, Kelurahan Cengkareng Barat, mengatakan kebakaran yang melanda  tiga ruko tersebut terjadi sekitar pukul 09.00. Api berasal dari salah satu ruko yang jadi bengkel motor.“Asal api dari bengkel motor yang montirnya sedang memperbaiki motor. Tapi penyebabnya saya kurang tahu,” ujar Nurdin, Senin (3/2).Saat api mulai berkobar, para montir langsung mengeluarkan motor yang ada di bengkel, sehingga tidak sampai ada motor yang ikut terbakar. Namun, imbasnya api merembet ke ruko yang ada di sebelahnya yang juga jadi bengkel motor.Kasudin Pemadam Kebakaran dan Penanggulangan Bencana (Damkar dan PB) Jakarta Barat, Jon Vendri menuturkan, untuk memadamkan api pihaknya mengerahkan sebanyak 28 unit mobil pemadam dengan rincian 20 unit dari Damkar dan PB Jakarta Barat dan 8 unit bantuan dari Dinas Damkar dan PB DKI. Berkat kerja keras petugas, sekitar pukul 11.00 api akhirnya bisa dipadamkan.“Yang terbakar tiga ruko yang jadi bengkel motor, yaitu ruko nomor 1 B, 1 C dan 1 D. Asal api diperkirakan dari lantai dua salah satu bengkel motor yang sampai saat ini masih kami data,” tandasnya.
