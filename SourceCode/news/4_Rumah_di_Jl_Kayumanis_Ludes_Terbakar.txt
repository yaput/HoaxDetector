4 Rumah di Jl Kayumanis Ludes Terbakar
Reporter : Nurito, Jhon Syah Putra Kaban, Yance Wiratman | Editor : Dunih | Selasa, 11 Maret 2014 17:45 WIB | Dibaca 628 kali
(Foto :  / Beritajakarta.com)
                    Sedikitnya empat rumah di Jl Kayumanis 3, RT 17/02 Kelurahan Kayumanis, Matraman, Jakarta Timur, ludes dilalap api, sekitar pukul 15.30. Tidak ada korban jiwa dalam peristiwa tersebut, namun warga korban kebakaran menderita kerugian materi hingga ratusan juta rumah.Dari empat rumah yang terbakar, di antaranya adalah rumah kost milik Ely (45) dan satu rumah mewah berlantai dua milk almarhum H Toto. Selain itu, api juga menghangsukan warung kelontong milik Mulyono (50). Api baru berhasil dipadamkan sekitar satu jam setelah kejadian menggunakan 20 unit mobil pemadam dari Sudin Pemadam Kebakaran dan Penanggulangan Bencana (Damkar dan PB) Jakarta Timur.Sekretaris RW 02 Kayumanis, Nugi Sanusi (50) menuturkan, api diduga pertama kali muncul dari rumah Ely yang dijadikan tempat kost. Api muncul dari lantai dua rumah tersebut. Saat kejadian sempat terdengar suara ledakan di lantai bawah, yang diduga berasal dari tabung gas. Tak lama kemudian api merembet ke bangunan lainnya yang ada di sampingnya. Termasuk rumah mewah dua lantai milik almarhum H Toto, ikut ludes terbakar."Tadi ada suara ledakan dua kali. Api pertama muncul dari lantai dua, kemudian merembet ke bawah dan bangunan yang ada di sekitarnya," ujar Nugi, Selasa (11/3).Warga yang panik langsung bahu-membahu memadamkan kobaran api. Namun sayang, upaya warga tak berhasil. Api terus berkobar hingga akhirnya merembet ke bangunan yang ada di sekitarnya.Kasie Operasi Sudin Pemadam Kebakaran dan Penanggulangan Bencana (Damkar dan PB) Jakarta Timur, Mulyanto mengatakan, dugaan sementara kebakaran dipicu oleh hubungan arus pendek listrik. Tak ada korban jiwa dalam peristiwa tersebut. Namun, kerugian materi yang ditimbulkannya mencapai ratusan juta rupiah."Kami kerahkan 20 unit mobil pemadam. Aksesnya sangat sulit, karena banyak warga berkerumun di sekitar lokasi kejadian. Mobil milik warga juga banyak parkir di jalan membuat laju kendaraan kami terhambat," katanya.Di Jakarta Selatan, kebakaran juga menghanguskan sebuah rumah di pemukiman padat penduduk Jl Muara RT 08 RW 03, Jagakarsa, Jakarta selatan, sekitar pukul 12.00. Rumah yang terbakar merupakan rumah dua lantai milik Mujito. Api yang berkobar selama satu jam akhirnya berhasil dipadamkan setelah 2 unit mobil pemadam datang ke lokasi kejadian.
