Warteg di Pegadungan Ludes Terbakar 
Reporter : TP Moan Simanjuntak | Editor : Widodo Bogiarto | Rabu, 04 Februari 2015 13:15 WIB | Dibaca 1043 kali
(Foto :  / Beritajakarta.com)
Sebuah rumah makan warung tegal (warteg) dan rumah kontrakan di Jalan Jambu Air RT 07/02, Kelurahan Pegadungan, Kalideres, Jakarta Barat, Rabu (4/2), ludes terbakar. Asal api diduga dari selang kompor gas yang bocor.

"  Kami sempat kesulitan memadamkan api karena banyak cairan kimia di lokasi kejadian"

Api pertama kali diketahui muncul sekitar pukul 10.30 dari dapur warteg milik Suryani (44). Tapi karena bangunan semi permanen membuat api cepat membesar dan menyambar bangunan rumah kontrakan yang ada di sebelahnya,â€ ujar Yudi, seorang saksi mata. Sebanyak 18 mobil pemadam kebakaran dikerahkan ke lokasi kejadian. Petugas dengan cekatan memadamkan api sehingga tidak merembet ke bangunan lainnya yang letaknya berdempetan. Api akhirnya berhasil dipadamkan sekitar pukul 11.30. "Kami sempat kesulitan memadamkan api karena banyak cairan kimia di lokasi kejadian," kata Pardjoko, Kepala Suku Dinas Penanggulangan Kebakaran dan Penyelamatan Jakarta Barat.
