Gudang Barang Bekas di Pulogebang Terbakar
Reporter : Nurito | Editor : Agustian Anas | Selasa, 03 Februari 2015 11:30 WIB | Dibaca 1325 kali
(Foto : Nurito / Beritajakarta.com)
Sebuah gudang barang bekas di Jl Komarudin Gang Sentosa RT 001/ 05 Pulogebang, Cakung, Jakarta Timur, ludes terbakar, Selasa (3/2).

"  Api berkobar sekitar pukul 03.00 dinihari tadi"

Informasi yang berhasil dihimpun, api mulai berkobar sekitar pukul 03.00. Saat kejadian, pemilik gudang, Wangunar (40) tengah tertidur pulas. Api mendadak muncul dari balik tumpukan barang-barang bekas yang disimpan di salah satu sudut lahan. Api dengan cepat membesar dan menghanguskan seluruh tumpukan barang bekas yang berada di atas lahan seluas 800 meter persegi tersebut."Api berkobar sekitar pukul 03.00 dinihari tadi," ujar Warji (35) warga setempat.Dikatakannya, warga sebenarnya sudah berupaya memadamkan kobaran api. Namun minimnya sumber air dan banyaknya barang yang mudah terbakar, membuat api sulit dipadamkan. Karena api semkain membesar, akhirnya warga menghubungi petugas pemadam kebakaran.Tak lama kemudian, sekitar 12 unit mobil pemadam dari Suku Dinas Pemadam Kebakaran dan Penanggulangan Bencana (Sudin Damkar dan PB) Jakarta Timur tiba di lokasi kejadian. Setelah berjibaku sekitar 4,5 jam, petugas akhirnya berhasil memadam  kan kobaran api."Api berhasil dipadamkan sekitar pukul 07.30," kata Moelyanto, Kasi Operasi Sudin Damkar dan PB Jakarta Timur.Dia mengatakan, tidak ada korban jiwa dalam kejadian tersebut. Namun seorang warga bernama Rohman (41) mengalami luka bakar di bagian kaki saat mencoba memadamkan kobaran api."Kerugian materi akibat kebakaran ini diperkirakan mencapai Rp 30 juta," ungkapnya.
