Panti Asuhan di Ciracas Ludes Terbakar
Reporter : Nurito | Editor : Dunih | Jumat, 20 Maret 2015 18:03 WIB | Dibaca 4843 kali
(Foto : Nurito / Beritajakarta.com)
Musibah kebakaran menghanguskan Panti Asuhan Bersinar di Jl Pembangunan, Kelapadua Wetan, Ciracas, Jakarta Timur, sekitar pukul 15.30. Dugaan sementara kebakaran terjadi akibat hubungan arus pendek listrik di salah satu kamar penghuni panti. Api baru berhasil dipadamkan sekitar 1 jam setelah kejadian menggunakan 10 unit mobil pemadam.

"  Api tahu-tahu sudah ada di atas genteng. Semua yang ada di dalam panti jadi panik dan berlarian keluar"

Unggul (48), guru di Panti Asuhan Bersinar mengatakan, saat sedang diadakan kegiatan belajar mengajar, api mendadak sudah di atap bangunan panti. Kondisi itu membuat 50-an anak didik di panti tersebut langsung berlarian keluar menyelamatkan diri.   "Api tahu-tahu sudah ada di atas genteng. Semua yang ada di dalam panti jadi panik dan berlarian keluar. Kemungkinan api berasal dari kamar penghuni panti," ujar Unggul, Jumat (20/3).Akibat kejadian itu, seluruh ruang belajar mengajar dan perpustakaan ludes terbakar. Termasuk bangku, meja, peralatan musik, peralatan tulis dan buku-buku bacaan.Tak hanya itu, api juga merembet ke salah satu rumah warga di Gang Baim RT 02/06 Kelapadua Wetan. Rumah tersebut berada persis di belakang Panti Asuhan Berinar. Beruntung 10 unit mobil pemadam cepat tiba di lokasi kejadian. Sehingga api dapat dilokalisir dan tak merembet ke rumah warga lainnya.Kasie Operasi Sudin Penanggulangan Kebakaran dan Penyelamatan Jakarta Timur, Moelyanto, mengatakan, dugaan sementara kebakaran dipicu hubungan arus pendek listrik. Namun, tidak ada korban jiwa dalam peristiwa itu. "Dugaan sementara akibat hubungan arus pendek listrik. Petugas kepolisian masih menyelidiki untuk mengetahui penyebab pastinya," tandasnya.
