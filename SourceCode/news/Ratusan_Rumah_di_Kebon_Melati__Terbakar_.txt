Ratusan Rumah di Kebon Melati  Terbakar 
Reporter : Rudi Hermawan | Editor : Lopi Kasim | Kamis, 05 Maret 2015 19:57 WIB | Dibaca 1871 kali
(Foto :  / Beritajakarta.com)
Peristiwa kebakaran kembali terjadi di Jakarta Pusat. Kali ini, ratusan rumah di pemukiman padat penduduk di Jalan Jati Bunder RT 14/14, Kebon Melati, Tanah Abang, Jakarta Pusat, hangus dilalap si jago merah. Diduga, penyebab kebakaran berasal dari korsleting listrik dari salah satu rumah warga. 

"  Tadi sekitar jam 16.30 api langsung membesar. Api berasal dari lantai 3 rumah kontrakan tepat di sebelah rumah saya"

Pantauan beritajakarta.com, sejumlah warga yang rumahnya terbakar terlihat histeris. Mobil pemadam kebakaran dari Sudin Pemadam Kebakaran dan Penanggulangan Bencana Jakarta Pusat yang dikerahkan ke lokasi tampak kesulitan mencapai titik api lantaran lokasi yang sempit. Hirmanto (31) salah satu warga yang rumahnya terbakar mengatakan, saat itu dirinya tengah bermain bersama anaknya terdengar teriakan adanya kebakaran. Ia pun kemudian langsung berusaha menyelamatkan keluarga serta harta bendanya. "Tadi sekitar jam 16.30 api langsung membesar. Api berasal dari lantai 3 rumah kontrakan tepat di sebelah rumah saya," katanya, Kamis (5/3).Surahmat, salah satu petugas Sudin Pemadam Kebakaran dan Penanggulangan Bencana   Jakarta Pusat mengatakan, untuk memadamkan api, pihaknya mengerahkan sebanyak 30 mobil damkar. Api kemudian dapat dijinakkan pada pukul 19.00. "Ada sekitar 30 mobil yang kami kerahkan dan 2,5 jam kemudian api baru dapat kami kuasai. Saat ini sedang dilakukan pendinginan," ujarnya.
